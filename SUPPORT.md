# Soporte y Contribución

¡Gracias por usar NetBreaker! Aunque el código fuente de este proyecto es cerrado, valoramos mucho el feedback de nuestros usuarios.

## Cómo obtener ayuda

1. **Revisa la documentación**: Lee el `README.md` y las Preguntas Frecuentes.
2. **Busca Issues existentes**: Alguien podría haber reportado ya tu problema.

## Reportar Bugs (Errores)

Si encuentras un error, abre un "Bug Report" en la pestaña de Issues. Asegúrate de incluir:
- Marca y modelo de tu dispositivo.
- Versión de Android.
- Versión de NetBreaker instalada.
- Pasos detallados para reproducir el error.
- Capturas de pantalla (si aplica, ocultando información sensible como tu IP pública).

## Solicitud de Funciones

¿Tienes una idea genial? Abre un "Feature Request" en Issues. Describe claramente qué funcionalidad te gustaría ver y por qué sería útil.

**Nota**: Pull Requests (PRs) con código no son aceptados ya que el repositorio no contiene el código fuente.
