# Política de Seguridad

Nos tomamos la seguridad de **NetBreaker** muy en serio. Apreciamos la contribución de la comunidad para mantener nuestra aplicación segura.

## Versiones Soportadas

Solo ofrecemos soporte de seguridad para la última versión mayor.

| Versión | Soportada |
| ------- | ------------------ |
| 3.x | ✅ |
| 2.x | ❌ |
| < 2.0 | ❌ |

## Reportar una Vulnerabilidad

Si descubres una vulnerabilidad de seguridad en NetBreaker, por favor **NO abras un Issue público**.

Envía un correo electrónico a `security@netbreaker.app`. Incluye:
1. Pasos para reproducir el problema.
2. Versión de la app y modelo de dispositivo afectado.
3. Impacto potencial.

Nuestro equipo responderá en un plazo máximo de 48 horas.
